## Ejecutar en desarrollo
1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Es necesario tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levatar la base de datos


## Tecnologias utilizadas
* Mongo DB
* Nest
